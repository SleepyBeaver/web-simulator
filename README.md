# Frontend – WEB-симулятор бизнес-процессов

Frontend-часть WEB-симулятора бизнес-процессов реализована на базе **React** с использованием **TypeScript** и сборщика **Vite**.

## Назначение

Клиентское приложение предназначено для обеспечения пользовательского интерфейса WEB-симулятора и решения следующих задач:

- визуальное моделирование бизнес-процессов;
- интерактивное редактирование элементов моделей;
- настройка параметров симуляции и сценариев выполнения процессов;
- отображение результатов анализа и имитационного моделирования.

## Архитектурные особенности

Frontend-приложение разрабатывается в виде одностраничного веб-приложения (SPA) и ориентировано на работу исследователя бизнес-процессов.

Архитектура построена на принципах:
- компонентного подхода;
- масштабируемости и расширяемости;
- разделения логики представления и бизнес-логики интерфейса.

Это позволяет в дальнейшем без существенных изменений подключать дополнительные UI-библиотеки, средства визуализации и инструменты аналитики.

## Используемые технологии

- React
- TypeScript
- Vite

# Backend

Backend реализован на NestJS и предназначен для:
- хранения моделей бизнес-процессов
- выполнения имитационной симуляции
- расчёта показателей эффективности
- формирования аналитических данных и отчётов

Архитектура построена по модульному принципу
с возможностью расширения функционала.
